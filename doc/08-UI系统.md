# 08 · UI系统

## 概述

FarmCreator 使用纯代码驱动的 UI 方案，所有面板由 `UISimplePanel.ts` 程序生成，无需预制体依赖。

- **核心文件**: `UISimplePanel.ts` (~1360行, 59KB)
- **管理器**: `UIManager.ts`
- **设计理念**: 单文件驱动、纯代码创建、背包风格视觉

---

## 一、架构设计

### 组件关系

```
GameManager.start()
  → ensureUIManager() 创建 UIManager 节点
    → UIManager.onLoad() 设置单例

点击商店/仓库节点
  → Shop.ts / Storehouse.ts
    → UIManager.getInstance().openShopPanel/WarehousePanel()
      → 创建面板节点 + UISimplePanel 组件
      → panel.initShop() / initWarehouse()
        → 创建背包风格面板 → 显示内容
```

### 关键API

```typescript
// UIManager
UIManager.getInstance().openShopPanel()     // 打开商店
UIManager.getInstance().openWarehousePanel() // 打开仓库
UIManager.getInstance().closePanel()         // 关闭面板

// UISimplePanel (单文件双模式)
panel.initShop()       // 商店模式：显示可购种子
panel.initWarehouse()  // 仓库模式：显示收获作物
```

## 二、场景配置

### GameManager 节点结构

```
Canvas
├── GameManager (GameManager组件)
├── CurrencySystem (CurrencySystem组件)
├── ShopManager (ShopManager组件)
├── WarehouseManager (WarehouseManager组件)
├── UIManager (UIManager组件)  ← 自动创建
├── shop (Shop组件)           ← 商店入口
└── warehouse (Storehouse组件) ← 仓库入口
```

### 执行顺序（Execution Order）

| 节点 | Execution Order | 原因 |
|------|----------------|------|
| CurrencySystem | -10 | 首先初始化 |
| WarehouseManager | -5 | 依赖货币系统 |
| ShopManager | -5 | 依赖货币系统 |
| UIManager | 0 | 最后，可依赖上述系统 |

## 三、功能详解

### 商店面板

- 显示所有可购买的种子（栅格布局）
- 支持"已解锁/未解锁"Tab切换
- 默认显示"已解锁"Tab
- 种子信息：名称/图标/价格/成熟时间/售卖价/解锁等级
- 异步重试机制：`isDataReady()` + 0.15s间隔重试×10次

### 仓库面板

- 显示已收获的作物（栅格布局）
- 每格显示：名称/图标/数量/单价/总价值
- 支持排序功能（按名称/数量/价格）
- 空仓库时显示美化提示（大图标+引导文字）

### 物品交互

- **点击格子** → 弹出详情弹窗（名称/价格/描述/属性）
- **数量选择器** → 弹窗内 +/- 调整数量，支持批量买/卖
- **选中高亮** → 金色边框脉冲动画 + 缩放回弹过渡

## 五、事件绑定

| 事件 | 触发源 | 监听者 | 用途 |
|------|--------|--------|------|
| `gold-changed` | CurrencySystem | 商店面板 | 更新金币显示 |
| `warehouse-changed` | WarehouseManager | 仓库面板 | 刷新仓库内容 |

双重事件注册防护：`isEventsRegistered` 标志防止重复监听。
