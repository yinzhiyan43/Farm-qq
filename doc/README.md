## 🔧 核心系统一览

| 系统 | 文件 | 职责 |
|------|------|------|
| 游戏入口 | `GameManager.ts` | 系统初始化、场景调度 |
| 作物系统 | `Crop.ts` | 生长周期、阶段管理、收获 |
| 土地系统 | `soil.ts` | 瓦片地图、扩建、交互 |
| 货币系统 | `CurrencySystem.ts` | 金币管理、持久化 |
| 仓库系统 | `WarehouseManager.ts` | 作物存储、出售 |
| 商店系统 | `ShopManager.ts` | 种子购买、解锁 |
| 经验系统 | `ExpSystem.ts` | 等级、经验、解锁 |
| 任务系统 | `TaskManager.ts` | 日常/主线任务 |
| 成就系统 | `AchievementManager.ts` | 8类成就、品质等级 |
| 时间系统 | `TimeSystem.ts` | 游戏时间、季节、暂停 |
| 天气系统 | `WeatherSystem.ts` | 5种天气、季节联动 |
| 好友系统 | `FriendsSystem.ts` | 好友列表、访问、送礼 |
| 存档系统 | `SaveManager.ts` | 多槽位、自动存档 |
| UI管理 | `UIManager.ts` + `UISimplePanel.ts` | 面板管理 |
| 动画系统 | `AnimationSystem.ts` | 收获特效、面板动画 |
| 性能系统 | `PerformanceSystem.ts` | 对象池、FPS监控 |
