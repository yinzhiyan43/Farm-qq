# 01 · 项目概览

## 项目概述

**FarmCreator** 是一个基于 Cocos Creator 3.8.7 开发的农场模拟经营游戏。

- **项目路径**: `F:/bigsinger/Farm/FarmCreator`
- **引擎版本**: Cocos Creator 3.8.7
- **开发语言**: TypeScript
- **目标平台**: Web (H5 小游戏)

## 项目背景

本项目参考了 SDL 农场游戏开发教程，使用 Cocos Creator 开发。历经多次引擎升级（1.9 → 3.8.7），当前为完整可玩的游戏 Demo。

## 核心系统

### 1. 作物系统
- 16种作物，每种独立生长周期（种子→发芽→生长→开花→结果→成熟）
- JSON 数据驱动配置（`crops.json`）
- 支持多年生作物、多生命周期

### 2. 经济系统
```
种植流程：
1. 商店购买种子 → 2. 点击土地种植 → 3. 等待成熟 → 4. 收获（金币+仓库）→ 5. 仓库出售
```

### 3. 货币系统（CurrencySystem）
- 金币增减、消费检查、自动持久化
- 单例模式，全局事件通知

### 4. 仓库系统（WarehouseManager）
- 作物存入/出售、价值计算
- 按作物ID分类，自动持久化

### 5. 商店系统（ShopManager）
- 种子购买、等级解锁机制
- 动态商品列表、解锁条件检查

### 6. 经验/等级系统（ExpSystem）
- 50级上限，指数级经验需求
- 等级解锁作物和功能

### 7. 任务系统（TaskManager）
- 日常任务/主线任务，进度追踪
- 奖励领取、每日自动重置

### 8. 成就系统（AchievementManager）
- 8大类型成就、5级品质（青铜→钻石）、35+成就项
- 进度追踪、自动解锁

### 9. 时间系统（TimeSystem）
- 游戏日/小时/分钟、季节变化
- 暂停/加速机制、存档恢复

### 10. 天气系统（WeatherSystem）
- 5种天气（晴/阴/雨/雪/风暴）
- 季节权重、作物生长影响、自动变化

### 11. 好友系统（FriendsSystem）
- 好友列表、访问农场、送礼、排行榜

### 12. 存档系统（SaveManager）
- 3个存档槽位、自动存档（1分钟间隔）
- 存档导入/导出、版本迁移

### 13. 事件总线（EventBus）
- 全局事件分发、100条历史记录
- 类型安全的15+事件类型

### 14. 动画系统（AnimationSystem）
- 作物摇摆、收获金币飞出、成熟指示器
- UI面板弹入弹出、Toast提示、按钮脉冲

### 15. 性能系统（PerformanceSystem）
- 对象池（CoinFly/ToastText/MatureIndicator）
- FPS监控、自适应降级、分帧更新

## 技术架构

### 单例模式管理器

```typescript
CurrencySystem.getInstance()   // 货币系统
WarehouseManager.getInstance() // 仓库系统
ShopManager.getInstance()      // 商店系统
UIManager.getInstance()        // UI系统
SaveManager.getInstance()      // 存档系统
```

### 数据持久化

使用 `localStorage` 自动保存，键名：
- `farm_gold` / `farm_diamond` — 货币
- `farm_warehouse` — 仓库数据
- `farm_shop` — 商店数据
- 各系统独立持久化 + SaveManager 统一存档

### 事件驱动架构

```
玩家操作 → EventBus.emit() → 各系统响应 → 触发UI更新 → 自动存档
```

## 使用说明

### 运行游戏
1. 在 Cocos Creator 3.8.7 中打开项目
2. 打开 BootScene 或 MainScene
3. 点击预览运行

### 基本操作
- 点击 **土地** — 种植作物
- 点击 **商店** — 购买种子（支持等级解锁）
- 点击 **仓库** — 管理收获物（支持出售）
- 点击 **扩展牌** — 扩建新土地
- 点击 **设置** — 音效/BGM开关
